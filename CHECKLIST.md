# Pre-Ship Checklist

- [ ] Change is within requested scope.
- [ ] No unrelated files changed.
- [ ] UI checked on desktop viewport.
- [ ] UI checked on mobile viewport.
- [ ] Keyboard navigation works for new/changed controls.
- [ ] Visual focus states are visible.
- [ ] No obvious browser console errors.
- [ ] Performance impact considered.
- [ ] Risks and limitations documented.
